⚠ Anti-jitter constraint
The live numbers above (mono-live) use fixed-width JetBrains Mono. At 15+ FPS, as digits change (e.g. 14 fps → 15 fps → 16 fps), the text column never reflows because each character occupies the same pixel width. Using Inter or any proportional font here causes visible horizontal jitter.
