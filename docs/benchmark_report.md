2. **dGPU Acceleration**: NVIDIA GeForce RTX 3060 Laptop GPU (dGPU) delivers significant acceleration for larger models and high tile counts compared to CPU execution.
3. **Variant Suitability**: Under the static image inspection timeline (budget: 10s), all model variants (Nano, Small, Medium) easily satisfy the NFR criteria on both CPU and dGPU, allowing edge operators to choose the higher mAP variants when dedicated accelerators are present.
