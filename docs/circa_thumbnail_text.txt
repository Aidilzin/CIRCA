CIRCA: Circuit Inspection and Recognition using Convolutional Architectures
Student: MUHAMMAD AIDIL AL-FAIZI BIN MOHD ZIN

CIRCA (Circuit Inspection and Recognition using Convolutional Architectures) is an AI-driven Printed Circuit Board (PCB) defect inspection desktop application engineered specifically for small-to-medium electronics repair facilities, hardware research labs, and independent technicians. Manual PCB inspection is notoriously error-prone, labor-intensive, and highly vulnerable to variable workbench lighting conditions. While industrial automated optical inspection (AOI) systems exist, their prohibitive cost makes them inaccessible for local repair centers. CIRCA addresses this operational gap by delivering high-precision computer vision defect detection directly on standard CPU workstations, eliminating the need for expensive GPU accelerators or specialized camera hardware.

Developed using the lightweight YOLOv12-Nano architecture and optimized with Intel OpenVINO INT8 post-training quantization, CIRCA executes static-image defect analysis in just 0.392 seconds per frame on standard CPU hardware. The model was domain-adapted using copy-paste synthetic dataset augmentation across 2,070 composite PCB images to reliably identify six critical IPC-A-600 defect categories: short circuits, open circuits, spurs, mouse bites, pin holes, and spurious copper. To overcome real-world environmental lighting variations, CIRCA integrates a real-time Auto-Optimisation vision pipeline (CLAHE contrast adaptive histogram equalization and HSV jitter compensation) that neutralizes workbench shadows and glare. Diagnoses are mapped to calibrated IPC Quality Grades (Class 3 High Reliability, Class 2 Dedicated Service, Class 1 General Electronic, or Inspection Fail) within a user-friendly PyQt6 interface complete with interactive scroll-wheel zoom, expandable plain-English defect definitions, and automated diagnostic ticket exports (TXT and CSV).

Empirical evaluation on frozen test splits demonstrates an impressive 85.70% Precision and 62.79% mAP@0.5. When deployed on 50 real-world phone-captured PCB test boards, CIRCA achieved a +600% True Positive recall lift (detecting 28 true defects compared to only 4 with un-tuned baselines). The integrated dynamic auto-optimisation pipeline incurs a minimal 73.1ms processing overhead while yielding an immediate +5.48% recall boost and +5.90% confidence gain under non-ideal lighting conditions. Designed for over 12,000 independent repair shops, CIRCA democratizes industrial-grade PCB quality assurance by reducing technician inspection fatigue, standardizing defect logging, and delivering a standalone, zero-cloud diagnostic ecosystem.
